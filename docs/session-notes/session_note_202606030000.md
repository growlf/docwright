## Key Technical Notes

- **`HUMAN_APPROVED=1`** env var bypasses the self-approval gate in both
  the pre-commit hook and the Claude Code PreToolUse hook.
- **`OPENCODE_URL`** env var switches `/api/overlap` from Jaccard to real LLM.
- **`DOCWRIGHT_ROOT`** env var overrides repo root for all server endpoints.
- `npm run test:dispatch` runs 34 tests outside the extension host (mocha + ts-node).
- Git panel "Stage all" calls `POST /api/git/stage` (git add -u — tracked only).
- The collation plan's Gantt view (Feature 3b) is deferred until `depends_on`
  fields are populated with real data.
