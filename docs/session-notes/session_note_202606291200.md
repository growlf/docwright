## Git Workflow Lesson (important)

**Correct GitFlow for this repo:**
- `feat/<name>` ‚Üí PR ‚Üí `develop` (collaboration; CI requires name matches feat|fix|chore|docs|refactor|test|policy|decision|dependabot/*)
- When releasing: `git checkout -b release/vX.Y.Z` from develop ‚Üí PR ‚Üí `main`
- Only `release/*` or `hotfix/*` branches may target main (enforced by Branch policy check CI)
- NEVER commit directly to develop or main
