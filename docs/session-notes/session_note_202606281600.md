## Decisions Made

- `AUTH_MODE=none` (default) bypasses all session logic — local dev unchanged
- `AUTH_MODE=local` uses `LOCAL_AUTH_USER` / `LOCAL_AUTH_PASSWORD` env vars — single admin, air-gap safe
- `AUTH_MODE=forgejo` — full Forgejo OAuth with team membership fetch
- In-memory session store (8h TTL, prune interval) — Redis adapter stubbed for Phase 3
- ETag is SHA-256 prefix of raw file bytes — deterministic, no clock-skew risk
- OCC guard is opt-in: writes without `If-Match` still succeed (backwards compat)
- `HUMAN_APPROVED=1` preserved in git commit env; GIT_AUTHOR_NAME/EMAIL now set from `locals.user` when authenticated
