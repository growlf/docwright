## Decisions Made

- **Keep TypeScript.** SvelteKit anchors the language choice; the dispatch module being importable by SvelteKit routes without an HTTP boundary is a real simplicity win. Python `mcp-server.py` migration is already paid off. RLM integration path is a Python microservice (~50 lines), not a language switch.
- **RLM is not a paradigm shift.** It is a better implementation strategy for multi-document AI tasks (collation, policy atom cross-reference) that are currently stubs. No plan rewrites needed. Integration depends on ai-stack GPU fix + policy atom framework being further along.
- **TRM/Mamba-2 (arxiv 2602.12078)** is primarily relevant to ai-stack/Meshy for future model selection, not DocWright directly. Linear-time SSM inference is worth tracking for iGPU-constrained nodes.
- **`research/INDEX.md` is a navigation file, not a research document.** Pre-commit hook updated to exclude it from frontmatter validation.
- **New rule:** Always look up local docs first, verify, then answer. Never assume file existence or conventions.
