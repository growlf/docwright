# Bug report should be modal

## Description

When filling in a bug/feature report, it should not be possible for the screen to reset or close the report modal - it is very frustrating to users to type in all the data for a report, only to lose it last second as the screen updates and removes it.  Annother option might be to just store the state (and content) so that it just comes right back to where they were with no loss of content but an updated screen.
