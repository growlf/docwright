## Problem

When creating or updating a proposal, we need a way of categorizing them and setting a guess on difficulty/complexity and time-to-build. This will help prioritize them and set hierarchy.

Some example categories might be (for a software engineering profile):

*   UI
*   UX
*   ENGINE
*   DATA

Occasionally, a proposal will fit into more than one category.
